# TericSoft

<h2>Install npm packages</h2> 
npm install

<h2>Run the server</h2>
npm start

<h2>Install JSON Server packages</h2>
npm i json-server

<h2>Start JSON Server</h2>
json-server --watch db.json --port 8080

<h2>View in Browser</h2>
http://localhost:3000

Or

https://teric-soft-sand.vercel.app/
